# joke
Orz 这个项目用来记录日常有趣的对话和内容，没事时候可以看看、乐乐，也能让新人更了解大家。

```
边边姐：「小心我的 U 盾攻击」（就是会不给发工资、不给报销，银行卡里啥也收不到。。。）
```

```

所以边边姐说一件事情，大家不知道，就会问俊文，
然后俊文一般的答复是：
「我 90 后也不清楚啊！」
```
